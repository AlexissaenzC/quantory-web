Evaluación

Ejercicios, cuestionarios y proyecto final

El curso de Python Básico introduce al participante en el mundo de la programación utilizando uno de los lenguajes más populares y versátiles de la actualidad. A lo largo del curso se construye una base sólida en la sintaxis del lenguaje, el razonamiento algorítmico y las estructuras fundamentales, permitiendo escribir programas funcionales y desarrollar el pensamiento lógico necesario para continuar en niveles superiores.

Objetivos de aprendizaje

  • Comprender los conceptos fundamentales de la programación y del lenguaje Python.
  • Instalar y configurar correctamente el entorno de desarrollo.
  • Manejar variables, tipos de datos, operadores y expresiones.
  • Aplicar estructuras de control de flujo para resolver problemas.
  • Utilizar las estructuras de datos básicas: listas, tuplas, diccionarios y conjuntos.
  • Definir y emplear funciones para organizar el código.
  • Desarrollar un programa completo de consola como proyecto final.

Contenido temático

Unidad 1. Introducción a Python
  • Historia y filosofía del lenguaje
  • Instalación y configuración del entorno
  • Primeros pasos: scripts, "Hola Mundo" y PEP 8
Unidad 2. Variables y tipos de datos
  • Variables, asignación y nomenclatura
  • Tipos primitivos, casting y entrada/salida de datos
Unidad 3. Operadores y expresiones
  • Operadores aritméticos, comparación y lógicos
  • Pertenencia, identidad, precedencia y f-strings
Unidad 4. Estructuras de control
  • Condicionales if / elif / else
  • Bucles while, for y control de iteraciones
Unidad 5. Estructuras de datos básicas
  • Listas, tuplas, diccionarios y conjuntos
  • Comprensión de listas (introducción)
Unidad 6. Funciones
  • Definición, parámetros y valores de retorno
  • Ámbito de variables y funciones lambda
Unidad 7. Cadenas, errores y archivos
  • Métodos de cadenas, búsqueda y reemplazo
  • Manejo de errores (try/except) y archivos de texto
Unidad 8. Proyecto final integrador
  • Diseño y desarrollo de un programa de consola
  • Buenas prácticas, presentación y revisión del proyecto
